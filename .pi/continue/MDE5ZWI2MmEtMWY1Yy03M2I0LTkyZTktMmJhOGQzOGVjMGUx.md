## Learned
- Use steer, not followUp, for pi-continue automatic mid-run resume dispatch because followUp can wait too long and leave the previous continuation still resuming. — source: previous-compaction-summary#Key.Decisions.pi-continue
- For VSCodium Claude Code under Nix-managed Home, preserve a versioned mutable compatibility path as a symlink to the managed extension rather than masking mutable generic binaries with nix-ld. — source: previous-compaction-summary#Key.Decisions.Claude-Code
- For disk cleanup, prune stale roots and reproducible build artifacts, then run GC; do not delete Nix store paths directly or remove current OS/user roots. — source: previous-compaction-summary#Key.Decisions.disk-cleanup
- Prometheus large-AI model weights are Nix-store managed through the current system generation; manual GC roots for current catalog models are redundant and can be removed when system roots are verified. — source: cmd:ssh root@prometheus remove manual roots#output system roots retained models_dir and presets
- Before deleting home-folder model downloads, compare basename candidates with Nix-store copies by sha256 and delete only exact duplicates. — source: cmd:ssh root@prometheus duplicate check#output two exact Qwen GGUF matches
- Whisrs currently ignores its documented audio device setting, so Home-level changes to [audio] device will not bind capture to DJI until Whisrs code changes or an environment-level PipeWire/ALSA target is used. — source: /git/github.com/LiGoldragon/whisrs/src/audio/capture.rs:43 and /git/github.com/LiGoldragon/whisrs/docs/configuration.md:45
- Do not restart PipeWire, WirePlumber, or Whisrs while whisrs status is recording; gather non-disruptive state first and wait for idle or explicit permission. — source: cmd:whisrs status#output recording during latest DJI debugging
- The DJI problem is no longer simply active profile flapping: the card can show headset-head-unit and the source can be RUNNING while the hardware path is silent until the profile is toggled. — source: user@latest-prometheus-dji-turn plus cmd:pactl/pw-cli latest DJI state
