# NOTA shape checklist

- Start from the expected type.
- Emit every positional value in order.
- Use vectors only for repeatable values and maps only for keyed values.

## NOTA literacy

- Follow the prompt's schema or examples.
- Emit positions in schema order without field labels.
- Return only the requested expression unless a diagnostic branch is provided.
