# NOTA shape checklist

- Start from the expected type.
- Emit every positional value in order.
- Use vectors only for repeatable values and maps only for keyed values.

## NOTA design

- Start from the expected type and positional shape.
- Use variants for alternatives and typed positions for optionality.
- Keep machine data in records and codecs.
