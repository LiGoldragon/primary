# 675 — The system, with perspective: frame and method

The psyche asked to see the whole system with lots of perspective — a wide-angle,
multi-zoom map that situates the just-landed criome internal policy language (the 674
arc, now on criome main via operator `signal-criome f10fb54d` / `criome cd1de18f`)
inside the entire intent → schema → component → deploy system.

This is a system-wide perspective, so it is a new report (675), not part of the
criome-specific 674 meta-report.

## Method — 5 grounding perspectives + 1 synthesis

Each grounding agent maps one perspective from real sources; the synthesis draws the
multi-zoom system map (the deliverable, highest-numbered file).

| File | Perspective | Grounded in |
|---|---|---|
| `1-telos-and-layers.md` | The telos + the vertical layer stack (intent → schema → component → deploy); psyche-primordial; today-vs-eventually | ESSENCE.md, INTENT.md, AGENTS.md, criome INTENT (eventual-Criome) |
| `2-component-constellation.md` | The components, each a triad (daemon + signal + meta-signal; Signal/Nexus/SEMA), and how they relate (criome auth, router transport, mirror vcs, spirit intent) | protocols/active-repositories.md, component-triad skill, per-repo INTENT/ARCHITECTURE |
| `3-schema-codegen-pipeline.md` | nota-next → schema-next (+schema-cc) → schema-rust-next → generated component types; schema as the single source of truth | the schema arc reports (666/667), nota/schema repos' INTENT |
| `4-e2e-and-deploy.md` | The d6he production chain (spirit → criome → router → mirror), the cluster (CriomOS, Yggdrasil, ouranos/prometheus), the two deploy stacks | reports 673, system-designer 123, CriomOS modules, INTENT two-stack discipline |
| `5-criome-language-in-context.md` | Where the just-landed policy language sits (criome's Nexus authorization), now on main | 674.15, operator 408/409, signal-criome/criome main |
| `6-system-map.md` | **The synthesis: the multi-zoom system map with diagrams at every level** | all of the above |

The synthesis is the wide-angle deliverable: a sequence of zoom levels, each with a
mermaid diagram and prose, from the telos down to the criome-language detail, closing
on where the system is right now.
