## Prompt frame

The psyche forwarded a prompt addressed to designer and asked operator for
an opinion with a subagent audit-opinion report. The forwarded prompt refined
the component shape around three runtime planes:

- **Signal** — reactive external input/output from people, agents, and
  messages.

  invokes state work.

objects and likely the trait surfaces; Rust implementation attaches behavior
as methods on those objects. The same prompt projected this shape into the
federated persona vision, where separate persona systems communicate through
skilled-agent tasks and context-bearing opinions.
