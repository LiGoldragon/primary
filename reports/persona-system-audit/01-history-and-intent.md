# Project History & Intent (corrected by psyche)

## Through-Line

The psyche (GitHub LiGoldragon) is building a system where AI agents do the cognition and everything deterministic is dumb typed machinery, aimed at software that self-improves toward the point where nothing better could be made. Continuous across predecessor and main line; only the substrate bet changed.

## Timeline & Why (psyche-confirmed)

- **Pre-history**: sema (2019, "save the biosphere with Sajban"), criomos-archive (2023). Old names circling the same idea.
- **Feb 18–20 2026**: agent-coding begins (117 commits Feb 20 in Mentci-AI). TRIGGER (psyche): discovered agent harnesses, which made the eventual sema/criome/persona end-forms buildable.
- **Predecessor (Feb–Apr)**: SEMA = a typed-binary format ("domain variants ARE the bytes, zero-copy") for LLM token-efficiency (claimed 2–3 orders of magnitude). aski = text DSL + 10-stage compiler to author SEMA. samskara = datalog/CozoDB agent (termination guarantee, ~12 constructs, closed-world). noesis = capnp RPC replacing MCP JSON-RPC. criome = agent runtime (network-namespace jails). Astrology baked into the ontology (Solar/Lunar polarity, 2→3→7→12→36→72→360 chain, zodiac-degree release tags, veri-core Astrality pillar).
- **THE RESET, psyche's own words**: aski was runaway complexity in character-level parser logic that couldn't carry an ambition that big; better to go straight to typed data in Rust backed by a database, no language layer. NOTA had to be created because JSON and similar are garbage formats. Two-staged: Apr 23 "a database is simpler than a parser pipeline"; Apr 25 "aski is dead / aski was the wrong way" — nexus rebuilt on NOTA.
- **CONTINUITY (psyche correction — important)**: aski was NOT abandoned in spirit. schema's SYNTAX still looks like aski. What changed is the APPROACH: schema uses structural/node-based parsing where the whole tree is typed structs resolving recursively via structural macros — NOT character-level parser logic ("if next char is X do Y"). The structural approach scaled to the ambition. Summary: aski-the-compiler was the wrong way; aski-the-syntax survived as schema.
- **NAMING (psyche)**: "aski" (A-S-K-I) was a pun on ASCII, mocking "we're still, ridiculously, dealing with text." "schema" = dropping the irony and getting serious about the ambition. Same syntax.
- **ASTROLOGY (psyche, "SROG"=STT error for astrology)**: NOT abandoned — dropped for context-inefficiency, to be reintroduced once the system is efficient enough to carry it. Deferred, not deleted.
- **MAIN LINE (Apr 23 →)**: Persona = a meta-AI system organizing models to emulate human intelligence, on the Criome stack. Three waves: (a) Apr 23–28 reset week (sema/signal/nexus/criome/CriomOS); (b) May 6–23 Persona component-daemon burst; (c) May 26–Jun 13 schema-derived wave (spirit + schema-next/nota-next/schema-rust-next).
- **mentci (Menchi)**: the USER INTERFACE for Persona (currently a generic criome approval UI; intended to grow into the Persona UI).
- **TELOS (psyche, articulated)**: AGI ("what people call artificial general intelligence") + the next crypto-based web. Persona = the intelligence; Criome = the crypto-native web.
- **META-OBSERVATION (psyche)**: "agents are really good at saying things are done that are not done" — verify completion independently.
